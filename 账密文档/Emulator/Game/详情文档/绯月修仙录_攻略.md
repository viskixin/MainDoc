# 常见问题

```markdown
# 右下角CG消失
修改目录下data文件夹，Map144.json文件 → "隐藏/打开地图立绘" : {"type":"false"}]} → 将false改为true即可

# 金羽箭不能使用
修改目录下data文件夹，Skills.json文件 → "中阶-金羽箭（上品）" : "occasion":2  → 将2改为1即可
注意：修改完之后，需要重新保存，再重新加载
```
