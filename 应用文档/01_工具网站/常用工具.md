```markdown
Update classes and resources
```

```tex
eclipse有红色叹号，一般情况可以这样处理：

如果eclipse项目有红色叹号，而Libraries中没有出现红色叉号的包，可以这样做：
	1) 先把eclipse转换到Markers视图，Window → Show View → Markers

Archive for required library: ‘XXXXXXXX001’ in project ‘xxxxxxxxx’ cannot be read or is not a valid ZIP file
	2) 用记事本打开项目根目录的.classpath文件，找到 <classpathentry kind="lib" path="XXXXXXXXX001"/> 删掉这一行
	保存，刷新后，这个报错就没有了，从而红色叹号也没有了。
```

#### * 关于 [%temp%](https://zhidao.baidu.com/question/15120339.html)

```tex
1) Temp
   电脑临时文件的归放处，这些临时文件一般存在时效性，过了时间段基本不会用，并且将该文件删除并不会对电脑造成影响
   操作：此电脑 → C:\Windows\Temp
                C:\Users\用户名\AppData\Local\Temp
   注意事项：不要删除文件夹本身
2) Prefetch
   系统运行产生的缓存垃圾，一般将里面的内容进行清理干净，就能节省不少空间
   操作：此电脑 → C:\Windows\Prefetch
   
3) WMI/setupcln
   日志记录文件夹，在使用系统文件或者电脑软件的时候，会进行处理记录，所以可以里面的一些文件进行删除
   操作：此电脑 → C:\Windows\System32\LogFiles

4) SoftwareDistribution
   经常有文件自动更新，该文件夹里面就是存放着自动更新时的安装包，所以更新完之后，文件夹里面的垃圾可以进行清理
   操作：此电脑 → C:\Windows\SoftwareDistribution
```
